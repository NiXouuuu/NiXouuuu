### About

<br/>

I run **KodiCore**, a European digital engineering company.  
Web development, custom automation, and game server infrastructure — for clients who need things done right.

My work sits at the intersection of clean product engineering and real-world systems performance.

Currently shipping:

- Full-stack SaaS products (React · Next.js · Supabase)
- FiveM server architecture & Lua systems
- Hosting infrastructure via OVH + Pterodactyl/Wings
- Automation & tooling for dev workflows
